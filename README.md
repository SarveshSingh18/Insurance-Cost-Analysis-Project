**Insurance Cost Analysis
Project Overview:**

This project analyzes the Insurance dataset to understand the key factors affecting medical insurance charges. Using exploratory data analysis and statistical methods, the project identifies relationships between demographic and lifestyle variables and insurance costs.



**Dataset Features:**

Age – Age of the individual

Sex – Gender of the individual

BMI – Body Mass Index

Children – Number of dependents

Smoker – Smoking status

Region – Residential region

Charges – Medical insurance cost



**Key Insights:**

Smoking has the strongest impact on insurance charges.

Higher BMI and age are associated with increased medical costs.

Regional differences have a relatively smaller influence on charges.



**Methods Used:**

Data Cleaning & Feature Engineering

Exploratory Data Analysis (EDA)

Pearson Correlation Analysis

Chi-Square Statistical Test

Data Visualization



**Technologies:**

Python

Pandas

NumPy

Matplotlib

Seaborn

SciPy

Jupyter Notebook


**
Conclusion:**

The analysis highlights how lifestyle and demographic factors influence insurance costs. These insights can help insurers design better pricing strategies and risk assessment models.
